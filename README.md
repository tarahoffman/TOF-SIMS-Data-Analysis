## Installation instructions


1. Install python
2. Download the repo
```commandline
git clone https://github.com/tarahoffman/TOF-SIMS-Data-Analysis.git
```
3. Create a virtual environment to install code in
```commandline
python -m venv venv
```
4. Activate the virtual environment
```commandline
venv\Scripts\Activate.ps1
```
5. Install packages
```commandline
pip install numpy matplotlib scikit-learn 
```


## Run instructions

```commandline
python .\src\SinglePCA.py
```
